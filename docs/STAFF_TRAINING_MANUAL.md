# Staff Training Manual - Payment System

## 1. Training Goal
Enable manager and cashiers to operate the system safely and confidently for daily supermarket sales.

## 2. Training Structure (2 Hours)
- Session A (Manager): 45 minutes
- Session B (Cashier): 45 minutes
- Session C (Simulation + Q&A): 30 minutes

## 3. Manager Training
## 3.1 Responsibilities
- Create and manage employee accounts
- Monitor inventory and low-stock alerts
- Check receipts and daily sales
- Review terminal status
- Review forecast and warning insights

## 3.2 Manager Workflow
1. Login with manager account.
2. Open management dashboard.
3. Review sales/receipts metrics.
4. Review low-stock list and plan restocking.
5. Verify terminals are active.
6. Check risk warnings from forecasting.

## 4. Cashier Training
## 4.1 Responsibilities
- Login at shift start
- Start sale, add products, verify totals
- Collect payment and issue receipt
- Handle cancellations using supervisor policy

## 4.2 Cashier Workflow
1. Login with cashier account.
2. Select store and terminal.
3. Start new sale.
4. Search and add products.
5. Confirm subtotal, tax, and total.
6. Choose payment method and process payment.
7. Generate receipt and confirm completion.

## 5. Standard Operating Rules
- Always verify product and quantity before payment.
- Never share login passwords.
- Report failed transactions immediately.
- Do not close browser during active payment.

## 6. Troubleshooting Guide
## 6.1 Cannot login
- Check email and password
- Ask manager to verify account active status

## 6.2 Product not found
- Check search text or barcode
- Ask manager to verify product setup

## 6.3 Payment failed
- Retry provider method once
- If still failing, use approved fallback method
- Notify manager

## 6.4 Receipt not generated
- Check cart has at least one item
- Confirm payment status is `paid`
- Notify manager

## 7. Employee Assessment Checklist
Manager should verify each cashier can:
1. Login and logout correctly
2. Start and complete a sale
3. Process payment and receipt
4. Explain what to do if payment fails

Pass rule: cashier completes 3 successful simulation transactions.
