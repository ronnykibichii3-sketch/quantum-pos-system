# Cashier Quick Reference Card

## Start of Shift
1. Open POS URL.
2. Login with cashier account.
3. Confirm store and terminal.
4. Start a new sale.

## Fast Sale Flow
1. Scan product barcode in scan field.
2. Confirm product appears in cart.
3. Repeat for all items.
4. Verify subtotal, tax, and total.
5. Click Pay.
6. Click Print receipt.

## If a Mistake Happens
- Wrong item: click Remove.
- Wrong quantity: use + / - buttons.
- Need to cancel sale: clear cart and start new sale.

## Common Problems
## Cannot Login
- Check email/password
- Ask manager to verify account is active

## Product Not Found
- Re-scan barcode
- Search by name
- Ask manager to check product setup

## Payment Error
- Retry once
- If still failing, notify manager

## Rules
- Do not share password.
- Always print or confirm receipt.
- Report system errors immediately.
