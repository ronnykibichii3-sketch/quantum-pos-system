# Pilot Acceptance Checklist

Use this during first-store go-live.

## Environment
- [ ] Server machine prepared
- [ ] `.env` configured
- [ ] Database reachable
- [ ] App starts without errors

## Functional
- [ ] Store created
- [ ] Terminal created
- [ ] Products created
- [ ] Manager employee created
- [ ] Cashier employee created
- [ ] Cashier login works
- [ ] Cart creation works
- [ ] Add/remove items works
- [ ] Payment works
- [ ] Receipt generated
- [ ] Stock updates after sale
- [ ] Reports reflect transaction
- [ ] Forecast generation works
- [ ] Warning panel displays output

## Operations
- [ ] POS accessible from cashier PC
- [ ] POS accessible from manager device
- [ ] Backup procedure tested
- [ ] Incident contact shared

## Signoff
- Store:
- Date:
- Developer:
- Manager:
- Result: PASS / FAIL
